# Document of a project

> 轻便 快捷 可靠

## clone
```
git clone git@git.arrway.cn:company/arrplat-docs.git
```
## init
```
cnpm install -g docsify-cli

```

## RUN
```
cd arrplat-docs    

# default port is 3000
#You can customize your port  
docsify serve
or
docsify s  


e.g.  
docsify serve <path> [--open false] [--port 3000]
```
